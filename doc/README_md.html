    Description¶ ↑

    Write a function that takes as its input a number (n) and converts it to a string, the contents of which depend on the number's factors: - if the number has a factor of 3, output 'Pling' - if the number has a factor of 5, output 'Plang' - if the number has a factor of 7, output 'Plong' - if the number does not have any of the above as a factor simply return the numbers digits
